promises :-----
==================
it is an object ,used to handle with asynchronus operation

states :=
   1. pending ----starting /initial states
   2.fulfilled --- if operation is success
   3.Rejected=== if operation is failed 
   promise can handle in two ways :
   1.  then and catch 
   then()---a function ,whihch is used to handle success of a promise 
   catch()---a function ,which is used to deal with failure of a promise 


   finally()--- a function,which executes even if we get success or failure 


   2. async and await 

   aync is a function which hold a promise
   await is a keyword used to wait for a promise to resolve
   await is used inside async function(mandatory )
